# PDF_RAG_CHATBOT


📚 PDF RAG Chatbot
Overview

PDF RAG Chatbot is a simple AI-powered application that allows users to upload a PDF document and ask questions based on its content. The application extracts text from the uploaded PDF and uses Google's Gemini AI model to generate relevant answers from the document.

Problem Statement

Users often need to search through lengthy PDF documents to find specific information. This project simplifies the process by enabling users to ask questions directly and receive answers based on the uploaded PDF content.

Features
Upload PDF documents
Extract text from PDF files
Ask questions related to the PDF content
Generate answers using Gemini AI
Simple and interactive Streamlit interface

Technologies Used
Python
Streamlit
Google Gemini API
PyPDF
Python-dotenv

Project Workflow
User uploads a PDF file.
Text is extracted from the PDF.
User enters a question.
PDF content and question are sent to Gemini AI.
Gemini generates an answer based on the PDF.
The answer is displayed in the chatbot interface.

Future Improvements
Multi-PDF support
Text chunking
Embeddings

Proper Retrieval-Augmented Generation (RAG)
